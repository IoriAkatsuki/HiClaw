## 完整命令说明

### 启动系统
```bash
./start_system.sh start
```
自动启动：
- WebUI HTTP服务器（端口8002）
- 统一检测程序（YOLO物体检测 + MediaPipe手部检测 + 激光标注）

### 停止系统
```bash
./start_system.sh stop
```
优雅地停止所有服务，清理PID文件

### 重启系统
```bash
./start_system.sh restart
```
等同于先stop再start

### 查看状态
```bash
./start_system.sh status
```
显示：
- WebUI服务器运行状态和PID
- 检测程序运行状态和PID
- 访问地址
- 日志文件位置

### 查看日志
```bash
# 查看所有日志
./start_system.sh logs

# 只看WebUI日志
./start_system.sh logs webui

# 只看检测程序日志
./start_system.sh logs detection
```

## 常见问题

### Q1: 启动失败，提示模型文件不存在
**A**: 检查模型文件是否在正确位置：
```bash
ls -lh ~/ICT/runs/detect/train_electro61/weights/yolov8_electro61_aipp.om
```
如果不存在，需要先训练或转换模型。

### Q2: WebUI无法访问
**A**: 检查步骤：
1. 确认WebUI服务器已启动：`./start_system.sh status`
2. 检查端口是否被占用：`ss -tulpn | grep 8002`
3. 确认防火墙未阻止8002端口
4. 尝试使用IP地址而非主机名访问

### Q3: 检测程序启动失败
**A**: 查看详细日志：
```bash
tail -f ~/ICT/logs/detection.log
```
常见原因：
- RealSense相机未连接
- Ascend环境未正确加载
- 模型文件损坏

### Q4: 视频流不更新
**A**:
1. 检查检测程序是否正常运行：`./start_system.sh status`
2. 检查RealSense相机：`rs-enumerate-devices`
3. 查看检测程序日志中的错误信息

### Q5: 手动停止进程后脚本无法启动
**A**: 清理残留PID文件：
```bash
rm -f ~/ICT/pids/*.pid
./start_system.sh start
```

### Q6: 如何修改危险距离阈值
**A**: 编辑start_system.sh，修改DANGER_DISTANCE变量：
```bash
nano ~/ICT/start_system.sh
# 找到 DANGER_DISTANCE=300，改为你需要的值（单位：mm）
```
然后重启系统：`./start_system.sh restart`

## 性能优化建议

### 提高帧率
1. 降低置信度阈值可能会检测更多物体但更快
2. 使用更小的模型（如yolov8n而非yolov8s）
3. 调整相机分辨率

### 降低资源占用
1. 限制最大检测手数（MediaPipe配置）
2. 增加推理间隔时间
3. 禁用不需要的功能

### 提高准确率
1. 提高置信度阈值（CONF_THRES）
2. 使用更大的模型
3. 优化NMS参数


### 自动启动（开机启动）
创建systemd服务：
```bash
sudo nano /etc/systemd/system/ict-laser-sorting.service
```

内容：
```ini
[Unit]
Description=ICT Laser Sorting System
After=network.target

[Service]
Type=forking
User=HwHiAiUser
WorkingDirectory=/home/HwHiAiUser/ICT
ExecStart=/home/HwHiAiUser/ICT/start_system.sh start
ExecStop=/home/HwHiAiUser/ICT/start_system.sh stop
Restart=on-failure

[Install]
WantedBy=multi-user.target
```

启用服务：
```bash
sudo systemctl daemon-reload
sudo systemctl enable ict-laser-sorting
sudo systemctl start ict-laser-sorting
```
