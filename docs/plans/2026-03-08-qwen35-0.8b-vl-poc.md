# Qwen3.5-0.8B 310B1 多模态 PoC Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在 Ascend 310B1 aarch64 板卡上，仅使用 Transformers + torch_npu + Hugging Face 原始权重，验证 Qwen/Qwen3.5-0.8B 单图单轮中文问答最小 PoC。

**Architecture:** 先在板卡侧做环境探测与依赖收敛，再用单文件 Python 脚本加载 processor 与模型，强制低分辨率、低上下文和 `max_new_tokens=128` 做一次推理，并通过 `/usr/bin/time -v`、`npu-smi` 采集 host/NPU 资源。若任一 unsupported op、dtype、OOM 或 processor mismatch 触发，则立即停止并生成失败报告。

**Tech Stack:** bash、Python 3、Transformers、torch、torch_npu、Pillow、huggingface_hub（仅用于已有权重路径/校验，不切换到其他推理框架）
